## Stock market data / Economic Data
- FRED API (Federal Reserve) - Extensive economic data, completely free

- World Bank API - Global economic indicators, free and well-documented

- IMF Data - International financial statistics, free access

- Yahoo Finance API (unofficial) - Stock/market data



## Buissness / Economic News 
- NewsAPI.org - Free tier: 100 requests/day, covers major financial news sources (Bloomberg, Reuters, CNBC, etc.). Good for headlines and article search.

- Currents API - Free tier with news from various sources including business/finance categories

- GNews API - Free tier: 100 requests/day, supports topic filtering for business news

- The Guardian API - Completely free, includes business/economics section

- New York Times API - Free tier: 500 requests/day, has business section access
